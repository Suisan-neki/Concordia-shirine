# 共有モジュール運用ルール

このディレクトリは、フロントエンドとバックエンドで共有する型・定数を置くための領域です。

## 置いてよいもの
- APIレスポンス/リクエストの共通型
- 画面とAPIで共通に使う列挙値・定数

## 置かないもの
- 実装ロジック（API呼び出し、DBアクセス、UIコンポーネント）
- 環境依存の値（URLやシークレット）

## 変更ルール
1. 共有型を変更した場合は、`client/` と `server_python/` の両方で影響確認を行う
2. 破壊的変更は、同一PR内で利用側の修正まで完了させる
3. 型の追加/変更には、必ずユースケース（どの画面/どのAPIが使うか）をコメントに残す
