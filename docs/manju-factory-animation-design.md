# お饅頭工場アニメーション設計

## コンセプト

「ヒューマンセキュリティなくしてサイバーセキュリティは実現しない」という概念を、食品工場のレーンのメタファーで表現する。

## ビジュアル構成

### 1. 左側エリア（ヒューマンセキュリティ工程）
- **人々のイラスト**: お饅頭を作る人々
- **状態による違い**:
  - 成功: 協力的に作業（笑顔、調和的な雰囲気）→ 綺麗な形のお饅頭
  - 失敗: 対立的に作業（険しい表情、ぎくしゃくした雰囲気）→ ぐしゃぐしゃのお饅頭

### 2. 中央エリア（ベルトコンベア）
- **レーン**: 横方向に流れるベルトコンベア
- **お饅頭**: レーン上を左から右へ移動
- **アニメーション**: 
  - ベルトの動き（点滅するライトや模様の移動）
  - お饅頭の移動（transform: translateX）

### 3. 右側エリア（サイバーセキュリティ工程）
- **包装機**: 機械的な装置のビジュアル
- **状態による違い**:
  - 成功: しっかり包装される（ラップで包まれる、保護される）
  - 失敗: 包装されない/雑な包装（むき出し、穴が空いている）

### 4. 最終地点
- **完成品の表示**:
  - 成功: 輝く、価値の高い製品
  - 失敗: くすんだ、価値のない製品

## シナリオ

### シナリオ1: 理想状態（ヒューマン◎ + サイバー◎）
1. 人々が協力的にお饅頭を作る
2. 綺麗な形のお饅頭がレーンに置かれる
3. レーンを流れる
4. 包装機でしっかり包装される
5. 輝く完成品として出荷される

### シナリオ2: ヒューマン欠如（ヒューマン× + サイバー◎）
1. 人々が対立的にお饅頭を作る
2. ぐしゃぐしゃのお饅頭がレーンに置かれる
3. レーンを流れる
4. 包装機でしっかり包装される
5. しかし中身が粗悪なので価値のない製品

### シナリオ3: サイバー欠如（ヒューマン◎ + サイバー×）
1. 人々が協力的にお饅頭を作る
2. 綺麗な形のお饅頭がレーンに置かれる
3. レーンを流れる
4. 包装機が機能せず、包装されない
5. 外気に触れて劣化し、価値のない製品

## 技術実装

### 使用技術
- React + TypeScript
- framer-motion（アニメーション）
- SVG（ビジュアル要素）
- Tailwind CSS（スタイリング）

### コンポーネント構成
- `FactoryAnimation.tsx`: メインコンポーネント
  - `HumanSecurityArea`: 左側エリア（人々とお饅頭生成）
  - `ConveyorBelt`: 中央エリア（レーンとお饅頭の移動）
  - `CyberSecurityArea`: 右側エリア（包装機）
  - `FinalProduct`: 最終地点（完成品）
  - `ScenarioControls`: シナリオ切り替えボタン

### アニメーションのタイミング
1. 人々の動作: 0-2秒
2. お饅頭がレーンに置かれる: 2秒
3. お饅頭がレーンを流れる: 2-5秒
4. 包装機の動作: 5-7秒
5. 完成品の表示: 7-8秒

### 状態管理
```typescript
type Scenario = 'ideal' | 'human-failure' | 'cyber-failure';

interface FactoryState {
  scenario: Scenario;
  isAnimating: boolean;
  currentStep: number; // 0-5
}
```
