# 参考画像の詳細分析

## コンベアベルトの構造
- **横レーン**：左から右へ（x = 120 ~ 500, y = 180）
- **L字の角**：(x = 500, y = 180)
- **縦レーン**：上から下へ（x = 520, y = 200 ~ 420）

## 左のアーム（1台目）の配置

### 参考画像での位置
- **ベース位置**：横レーンの**左上**
- **方向**：上から下に向かって伸びる（垂直下向き）
- **作業対象**：横レーンを移動中のお饅頭

### 推定座標
- `top: 100px`（ベルトの上側）
- `left: 200px`（横レーンの左側）
- `transform: rotate(0deg)`（回転なし、垂直下向き）

## 右のアーム（2台目）の配置

### 参考画像での位置
- **ベース位置**：縦レーンの**右側**
- **方向**：右から左に向かって伸びる（水平左向き）
- **作業対象**：L字の角～縦レーンを移動中のお饅頭

### 推定座標
- `top: 280px`（縦レーンの中央付近）
- `left: 640px`（縦レーンの右側）
- `transform: rotate(-90deg)`（反時計回りに90度回転、水平左向き）

## 重要なポイント

1. **2つのアームは異なる位置に配置**
   - 左のアームは横レーンの左上
   - 右のアームは縦レーンの右側
   - **並行配置ではない**

2. **ベルトに沿った配置**
   - 左のアームは横レーンの上側
   - 右のアームは縦レーンの右側
   - L字のベルトに沿って配置

3. **作業エリアの分担**
   - 左のアームは横レーンで包装紙を引く
   - 右のアームは縦レーンでリボンを付ける
