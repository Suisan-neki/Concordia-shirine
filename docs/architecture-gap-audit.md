# 体系差分メモ（設計/実装の不整合）

このメモは、現状の実装とドキュメントのギャップを整理するための簡易一覧です。

## 主な不整合
- `docs/architecture.md` が **tRPC + Lambda** を前提としているが、実装は **FastAPI (`server_python/`)** を中心に動いている
- ルート配下に `server_python/` と `src/` が共存し、役割が不明瞭（`src/` は音声プロトタイプ用のローカル実行コード）
- READMEのセキュリティ説明に **AI時代の脅威対策** などが含まれるが、UI/チュートリアルでは削除済み

## 影響
- 新規参画者が正しい構成を把握しにくい
- どのコードが現在利用されているのか判断が難しい

## 次の対応方針
- アーキテクチャ文書とREADMEを **現状実装に合わせて更新**
- `src/` の位置づけを明記し、**利用範囲と責務を明文化**
