# L字型レイアウト設計

## レイアウト構成

```
[作業員]  →  [横レーン（お饅頭が流れる）]  →  [角]
                                              ↓
                                         [縦レーン]
                                              ↓
                                    [サイバーセキュリティボックス]
                                         （包装される）
                                              ↓
                                          [消える]
```

## 座標設計

### 横レーン（上部）
- 開始位置: 作業員の手元（左側）
- 終了位置: 右端の角
- 長さ: 画面幅の約60%

### 縦レーン（右側）
- 開始位置: 角（横レーンの終端）
- 終了位置: 下端
- 長さ: 画面高さの約60%

### サイバーセキュリティボックス
- 位置: 縦レーンの中央付近
- サイズ: 縦レーンの幅より少し大きい正方形

## アニメーションパス

### お饅頭の移動経路

1. **フェーズ1: 作業員の手からレーンへ（0-1秒）**
   - 作業員の手が下がる動作
   - お饅頭がレーンに置かれる
   - opacity: 0 → 1

2. **フェーズ2: 横レーンを移動（1-4秒）**
   - x座標: 左端 → 右端
   - y座標: 固定
   - 直線移動

3. **フェーズ3: 角で曲がる（4-4.5秒）**
   - 90度回転
   - x座標: 固定
   - y座標: 開始

4. **フェーズ4: 縦レーンを移動（4.5-7秒）**
   - x座標: 固定
   - y座標: 上 → 下

5. **フェーズ5: サイバーセキュリティボックス通過（5.5-6秒）**
   - ボックス内に入る
   - 包装される（ラップが追加される）
   - ボックスから出る

6. **フェーズ6: 消える（7-7.5秒）**
   - opacity: 1 → 0
   - scale: 1 → 0.8

## 作業員の動作

### 協力的な作業員（理想状態、サイバー欠如）
- お饅頭を丁寧に持つ
- 手を下げてレーンに置く
- 笑顔

### 対立的な作業員（ヒューマン欠如）
- お饅頭を雑に持つ
- 手を下げてレーンに置く（同じ動作）
- 険しい表情

## サイバーセキュリティボックスの動作

### 正常動作（理想状態、ヒューマン欠如）
- お饅頭が入る
- 包装アニメーション（ラップが巻かれる）
- 包装されたお饅頭が出る

### 故障（サイバー欠如）
- お饅頭が入る
- 包装されずにそのまま出る
- 警告マーク表示

## レスポンシブ対応

### デスクトップ
- L字型レイアウト
- 横レーン: 長め
- 縦レーン: 長め

### タブレット
- L字型レイアウト（少し小さめ）

### モバイル
- L字型レイアウト（縦長に調整）
- または、上から下への直線レイアウトに変更
